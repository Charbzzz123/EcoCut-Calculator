# Brand Assets

Place production-ready art files here so the Angular app can reference them.

Required filenames:
- eco-logo.png: full-color EcoCut wordmark (used in the home hero).
- eco-mascot.png: character illustration with transparent background (optional; mascot-placeholder.svg is used if missing).

All assets are copied via public/assets during 
g build. Replace the placeholder art anytime without touching code.
